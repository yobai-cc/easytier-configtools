# Changelog

## 0.2.0 - 2026-04-02

### 新增
- 增加了粘贴式 TOML 导入流程，可将当前已支持的 EasyTier 字段回填到表单
- 增加了 `proxy_networks` 和 `port_forwards` 的结构化编辑
- 增加了面向源码交付的发布文档整理

### 改进
- 让生成出的 TOML 更贴近 EasyTier 官方常见结构
- 加强了 `peer` URI 校验和表单安全性
- 清理了仓库边界，使第一版源码交付更干净、更易移植

### 暂未包含
- 本版本不包含线上托管部署
- 本版本不包含 Docker 打包或桌面安装包
- 当前 UI 仍未覆盖全部 EasyTier 配置选项
